# EV Charging Station OS Simulation ⚡

## Overview
This project is a comprehensive software simulation of an Electric Vehicle (EV) charging station operating system. Written entirely in C, it demonstrates advanced system architecture and dynamic memory management by simulating the operational logic, user interactions, and resource allocation of a modern charging network.

## Architecture & Design
The project is built using a **multi-file structure**, ensuring modularity, clean code, and separation of concerns. The core logic relies on advanced data structures to handle data efficiently:

*   **Linked Lists:** Utilized for managing dynamic queues (e.g., vehicles waiting for an available charging spot).
*   **Binary Trees:** Implemented for fast and efficient searching, sorting, and indexing of registered users and station IDs.
*   **Stacks:** Used to manage operational history, allowing for efficient state tracking and sequential execution of recent commands.

## Key Features
*   Simulated user authentication and session management.
*   Dynamic allocation of charging resources.
*   Real-time queue and station status tracking.
*   Modular codebase allowing for easy scalability and future feature integration.
